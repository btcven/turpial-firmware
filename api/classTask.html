class Task¶

Encapsulate a runnable task.

This class is designed to be subclassed with the method:

void run(void *data) { ... }

For example:

class CurlTestTask : public Task {
   void run(void *data) {
      // Do something
   }
};

implemented.

Subclassed by esc::Interrupt
